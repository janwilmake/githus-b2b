Frequently Asked Questions

What is GitHus and how does it differ from GitHub?
GitHus is a code context intelligence platform that works alongside GitHub to transform your repositories into a structured, queryable knowledge base. While GitHub stores your code, GitHus analyzes it to create business intelligence, enabling semantic search, SQL analytics, and contextual insights across your entire codebase.
How does the Enterprise plan pricing work?
Enterprise pricing starts at $5,000/month and scales based on the volume of code data points processed and LLM usage. Factors that influence pricing include the number of repositories, frequency of updates, historical data retention needs, and advanced features usage. Our sales team will work with you to create a custom plan that fits your organization's specific requirements.
What kind of insights can I get with SQL access to my code?
With direct SQL access to your code intelligence database, you can run complex queries to discover patterns across your codebase such as dependency usage, code quality metrics, developer productivity, technical debt indicators, security vulnerability patterns, and much more. You can build custom dashboards and integrate with your existing BI tools to gain unprecedented visibility into your software development lifecycle.
How does GitHus handle security and privacy?
GitHus takes security and privacy extremely seriously. We never store your actual source code - only the processed context and metadata. All data is encrypted at rest and in transit. Enterprise plans include advanced security features such as VPC isolation, custom retention policies, and compliance certifications. We can also deploy in your own environment for maximum security control.
Can I integrate GitHus with my existing development workflow?
Absolutely. GitHus is designed to complement your existing tools and workflows. We provide integrations with popular IDEs, CI/CD pipelines, project management tools, and business intelligence platforms. Our API allows for custom integrations with any systems specific to your organization. Enterprise customers receive custom integration support from our solutions team.
